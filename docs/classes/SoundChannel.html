SoundChannel Class

Defined in: SoundChannel:933
Module: xc.createjs.soundjs

An internal class that manages the number of active SoundInstance instances for each sound type. This method is only used internally by the Sound class.

The number of sounds is artificially limited by Sound in order to prevent over-saturation of a single sound, as well as to stay within hardware limitations, although the latter may disappear with better browser support.

When a sound is played, this class ensures that there is an available instance, or interrupts an appropriate sound that is already playing.
